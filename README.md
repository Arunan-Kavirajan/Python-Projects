# 🎯 Learning Philosophy

Instead of following tutorials and moving on, every new topic is reinforced by building projects.

The rules I follow are simple:

- ✅ Learn a new concept
- ✅ Build projects using only concepts covered so far
- ✅ Avoid project tutorials and copy-pasting
- ✅ Push every completed project to GitHub
- ✅ Focus on understanding rather than memorization

As I continue learning Python, this repository will gradually evolve from beginner console applications to larger real-world software projects.
